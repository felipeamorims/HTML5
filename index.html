Anotações sobre HTML

Sobre a página

Página criada para documentar o que fui aprendendo durante o curso da Udemy.

Doctype

Doctype significa tipo de documento, no caso html5 é apenas:

Tag: < !DOCTYPE html>

Pode ser que em outros sites encontraremos códigos gigantes dentro de doctype para definir outras versões, anteriores ao html5.

HTML

Tag que define onde começa o código html e onde termina. Dentro dela é onde iremos escrever nossos códigos, abrimos e fechamos ela.

Tag: < html > < /html >

Cabeçalho

Temos duas estruturas principais quando criamos código html, head e body.

Aqui dentro do Head (que pode ser omitido), colocamos estruturas que não aparecem tão visíveis ao site para o usuário mas são tão importantes quanto, geralmente são configurações da página. Podemos colocar estruturas de formatação colocar códigos javascript para animações no site ou interação com usuário.

Tag: < head > < /head >

Título

Título da página.

Tag: < title > < /title >

Meta

Alguns navegadores podem não exibir corretamente acentuações, para isso utilizamos

Tag: < meta charset="UTF-8" />

Dentro da tag podemos definir atributos (a codificação que iremos utilizar), que são informações adicionais atribuídos aquela tag.

Corpo

Aqui dentro do body colocamos informações que aparecem diretamente ao usuário.

Tag: < body > < /body >

Linha Horizontal

Horizontal Rule, separa duas seções maiores de texto com uma linha

Tag: < hr />

Parágrafo

Parágrafo pega várias linhas e poe dentro de um paragrafo. E sempre tem um espaçamento

Tag: < p > < p/>

Outros

Comentários são feitos com: < !-- texto comentado aqui -->

Self Closing Tag: "meta" possui um atributo chamado "charset".

Formatações de texto

Os textos não são formatados de acordo com espaçamentos no hora da programação. Todos os tipos de formatação precisam de códigos para funcionar, como por exemplo, paragrafos, espaços, quebras de linhas.

Como recomendação utilizar a tag "< strong > < /strong >", ao invés de "< b > < /b>", para fazer textos em negrito. Explicação aqui.

Como recomendação utilizar a tag "< em > < /em >", ao invés de "< i > < /i >", para fazer textos em itálico.

Para sublinhar utiliza-se a tag < u > < /u>.

Para tachar utiliza-se a tag < strike > < /strike>

.

Listas não Ordenadas

Aparecem com ponto para listar os itens, para criá-la utiliza-se a tag < ul> < /ul> . Além disso devemos criar os itens desta lista, utilizando a tag < li> < /li> .

Dentro das listas, podemos mudar o tipo de marcadores, para as não ordenadas basta utilizar desta forma incluindo "type" < ul type="circle"> < /ul>.

Listas Ordenadas

São listas que possuem contagem dos itens. Elas possuem marcador de numeração. Para criá-la utiliza-se a tag < ol> < /ol> . Além disso devemos criar os itens desta lista, utilizando a tag < li> < /li> .

Dentro das listas, podemos mudar o tipo de marcadores, para as ordenadas basta utilizar desta forma incluindo "type" < ol type="A"> < /ul>.
